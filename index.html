1. 아래 코드를 <head></head> 사이에 넣는다.
<script src="https://code.jquery.com/jquery-latest.min.js"></script>
<script type="text/javascript" src="https://garlicchicken.github.io/shEditor/shEditor.js"></script>

2. shEditor를 사용할 <form></form>사이에 <script></script>으로 shEditor 함수를 호출한다.
첫 번째 인자는 <form></form> id를 두 번째 인자는 백엔드로 전달할때 사용할 name을 입력한다.
<form id="thisForm">
  <script type="text/javascript">
    shEditor('thisForm','thisInputName'); // shEditor(form id, input name)
  </script>
</form>
